# Hypedrive Brand Assets

Public brand assets for Hypedrive. Served via jsDelivr CDN.

## Logos

| Asset | CDN URL |
|-------|---------|
| Full Logo (Icon + Text) | `https://cdn.jsdelivr.net/gh/ShivamGupta-SM/hypedrive-assets@main/logos/logo-full.svg` |
| Icon Only | `https://cdn.jsdelivr.net/gh/ShivamGupta-SM/hypedrive-assets@main/logos/logo-icon.svg` |

## Usage

### In HTML/Email
```html
<img src="https://cdn.jsdelivr.net/gh/ShivamGupta-SM/hypedrive-assets@main/logos/logo-full.svg" width="150" height="36" alt="Hypedrive" />
```

### Pin to specific version
```
https://cdn.jsdelivr.net/gh/ShivamGupta-SM/hypedrive-assets@{commit-sha}/logos/logo-full.svg
```
